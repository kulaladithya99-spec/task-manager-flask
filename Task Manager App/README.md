# Task Manager App

A full-stack task management web app built with Flask and Python.

## Live Demo
https://task-manager-flask-qpgq.onrender.com

## Features
- User registration and login with password hashing
- Add, edit, delete and complete tasks
- Organise tasks by categories
- Set due dates — overdue tasks highlighted
- Filter by status, priority and category
- Search tasks by title
- Dashboard with live stats and progress bar
- Profile page and account settings

## Tech Stack
- Python / Flask
- Flask-SQLAlchemy (SQLite)
- Flask-Login
- Jinja2 templates
- Werkzeug security

## Run locally
git clone https://github.com/kulaladithya99-spec/task-manager-flask
cd task-manager-flask
pip install -r requirements.txt
python run.py
